# Developer For Hire

One-page website for a local small-business web development offer.

## Run locally

```powershell
npm.cmd run serve
```

Then open `http://localhost:4173`.

## Check syntax

```powershell
npm.cmd run check:syntax
```

## Positioning

Developer For Hire builds fast custom websites that small businesses own. Static brochure sites do not require a mandatory monthly developer subscription; ongoing changes, maintenance, and dynamic features are scoped separately.

## Notes

The interactive 3D preview uses Three.js pinned in `package.json` and a local browser module in `vendor/three.module.min.js`, so the live page does not depend on a CDN.
